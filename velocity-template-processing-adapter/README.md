# Velocity Templates

## Options

Options kann be set via properties in the model YML file.

### Model

#### Attribute

##### `key-generation` (Optional)

**Layer:**
* jpa-persistence

**Values:**
* `sequence`

Marks an id element as to generate with sequence annotations.


##### `id` (Optional)

**Layer:**
* jpa-persistence

**Values:**
* None (set `true`)

Shows, that the attribute is a key attribute in the database.
